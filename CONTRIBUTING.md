# Contributing to Vue Select Plus

Thank you for your interest in contributing to Vue Select Plus!

## Getting Started

This project is a monorepo managed by NPM Workspaces.

### Prerequisites

- Node.js (Latest LTS recommended)
- NPM 7+ (comes with Node 15+)

### Setup

1. Fork and clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

## Development

### Running the Playground

We use a Playground to test changes in real-time.

```bash
npm run dev -w playground
```

### Running Documentation

```bash
npm run dev -w docs
```

### Building Packages

To build all packages:

```bash
npm run build
```

## Testing

We use Vitest for unit testing.

```bash
npm test
```

## Pull Request Process

1. Create a new branch for your feature or fix.
2. Ensure all tests pass.
3. Submit a Pull Request with a clear description of your changes.

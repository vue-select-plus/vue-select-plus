# @vue-select-plus/core

The headless core logic for **Vue Select Plus**.

> Contains the composables and state management logic.

## Installation

```bash
npm install @vue-select-plus/core
```

This package is usually installed automatically by `@vue-select-plus/vue`.
Use this if you are building a custom UI from scratch.

For full documentation, visit [vue-select-plus.github.io](https://vue-select-plus.github.io/vue-select-plus/).

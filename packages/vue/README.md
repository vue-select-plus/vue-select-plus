# @vue-select-plus/vue

The official Vue 3 component for **Vue Select Plus**.

> Enterprise-grade, headless, and accessible select component.

## Installation

```bash
npm install @vue-select-plus/vue
```

## Setup

Don't forget to import the styles!

```vue
<script setup>
import { VSelect } from '@vue-select-plus/vue';
import '@vue-select-plus/styles'; // Import global styles
</script>
```

For full documentation, visit [vue-select-plus.github.io](https://vue-select-plus.github.io/vue-select-plus/).
